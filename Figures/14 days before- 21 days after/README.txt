Directory containing figures generated over the timespan 14 days before the first initial case to 21 days after that case.

Cost Heatmaps/ -- Contains several different heatmaps containing cost function evaluations
R0 Heatmaps/ -- Contains several different heatmaps containing R0 value evaluations

Asymptomatic vs. Symptomatic Time Series -- Shows the distribution of symptomatic vs. asymptomatic cases
Beta_i heatmap -- A heatmap of beta_i values for different k and c values
Curve Fit Time Series -- A time series plot showing the fit of the model against the actual case data
Total Cases Heatmap: B_i=2 -- A heatmap showing the total case numbers with B_i fixed at 2.
